# thecodergabe-studio

A Nuxt 4 + Vuetify project with a clean SCSS architecture, custom variables, and utility classes.  
Configured for modern frontend development with Vite, Vue 3, and Nitro.

## Features

- Nuxt 4 application structure
- Vuetify 3 with auto-imported components
- Global SCSS (variables + utilities)
- sass-embedded for modern Sass compilation
- Clean alias resolution and project root configuration
- Hot module replacement via Vite
